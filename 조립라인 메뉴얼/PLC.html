PLC 관련
  • PLC 모니터링

    • 1.Work2 깔려있는 컴퓨터와 PLC연결. (이더넷 or 미니USB 사용)
    2.컴퓨터측 네트워크 확인 하여 PLC 네트워크안에 들어오게 해줌. IP4주소의 앞에 3부분 xxx.xxx.xxx 이 부분까지 같게 하고 맨끝 번호 xxx만 PLC와 다르게 바꿔줌.
    3.Work2 프로그램 열어준 후 상단 바 Read Online 실행. 파라메터 빼고 execute 버튼 누름.
    *최초 연결 시 연결 방식 설정 후 Test connection 누르고 확인 후 닫기.
    4.연결 후 F3 키 누르면 모니터링 시작 (*Shift + F3은 런중 수정 가능한 모니터링)

  • 작동 안되는 PLC Y접점 수정(런중수정)
    • 1.수정 필요한 래더맵에 위치
    2.Shift + F3 누름. 런중 수정 가능 모니터링 상태
    3.작동 안되는 Y접점을 다른 곳으로 옮겨주고 번호 확인.
    4.작동 안되는 Y접점을 그대로 두고 밑에 작동되는 코일을 추가 (나중에 다른 작업자가 보고 어떤 것이 작동안되는지 알수있게)
    5.완료 후 F4(컴파일) 누르고 마무리
